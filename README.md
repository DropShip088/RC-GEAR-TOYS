# RC GEAR TOYS Catalog

This catalog is now a Vite website.

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the final website:

```bash
npm run build
```

## Update Products

Product data is in `src/products.js`. Edit product names, prices, size, color, descriptions, and image URLs there.

Discounted selling prices are currently shown from Rs. 699.00 to Rs. 1,499.00 as requested.

## WhatsApp Checkout

The checkout link sends customers to:

`https://wa.me/918988520742`

The message includes the cart number, product list, quantities, total, and a request to confirm availability, delivery charges, and payment details.

## Security Notes

This is a Vite-powered static catalog. It does not collect card details, login details, or customer payment information. For live hosting, use HTTPS hosting and keep payment confirmation on WhatsApp.
